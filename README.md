# eKhayaToons

eKhayaToons is an African digital comic streaming and publishing platform designed for readers, creators, and communities.

## Features
- Comic Reader
- Creator Studio
- AI Chatbot
- Community Discussions
- Admin Dashboard

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS
- GitHub Actions

## Security
- CodeQL
- Dependabot
- npm audit
- ESLint
